Executive Summary

Overall Rating: 9.1 / 10

InvoraLite is a mature v1.0 offline desktop inventory and retail application for a single Windows store. Coverage includes sales, purchases, Rate Master, 20-type Offers, Quotations, credit, GST invoices, analytics, accounting, two-stage returns, purchase history, supplier ledgers, and a loyalty points MVP.

This late-evening recheck closes several prior quality gaps: GitHub Actions CI, Playwright E2E smoke scaffolding, vault unclean-exit / logout seal hardening, and customer loyalty earn/redeem.

Automated checkup (this run): 169 / 169 Vitest, 30 / 30 Rust, 2 / 2 Playwright smoke, TypeScript clean. Ready for trusted single-store deployment. Before wide public distribution: rotate license secrets and code-sign the installer (relational schema and auto-update remain future work).
